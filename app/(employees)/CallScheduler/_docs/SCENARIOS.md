# CallScheduler — 페르소나·시나리오

> CLAUDE.md 규칙 26 의무. 데이터 모델 / UI / API 결정 전 시나리오 워크-스루.
> 각 페르소나가 처음부터 끝까지 운영하는 흐름.

## 페르소나 1 — 매니저 (작성자)

### 1-A. 첫 월 셋팅 (모듈 도입 시)

```
Step 1.  /CallScheduler/settings → ⏰ 시간 (시프트) 탭
         · 13개 시프트 마스터 점검 / 추가
Step 2.  → 🧑‍🤝‍🧑 그룹 탭
         · 시프트 그룹 정의 (예: "주간 09-18", "야간 20:30-08:30")
         · 각 그룹에 멤버 + 패턴 (평일/주말/매일/요일지정) + 전략 (전원/로테이션)
Step 3.  → 👥 직원 (콜센터) 탭
         · RideEmployees 미활성화 후보 확인 → 콜센터 워커로 전환
         · 색상 토큰 / 그룹 라벨 조정
Step 4.  → 🏖 공휴일 (참고) 탭
         · 2026 공휴일 시드 확인 (15개 자동 들어감)
Step 5.  → 📋 직원 휴가 탭
         · [💼 일괄 발급]: 전 직원에게 연차 15일, 패밀리데이 월 1일 자동 발급
```

### 1-B. 월별 스케줄 작성

```
Step 1.  /CallScheduler 목록 → [+ 새 월 만들기]
Step 2.  연/월 선택 → [생성] (또는 전월 패턴 복제)
Step 3.  상세 페이지 → [✍️ 작성] 모드
Step 4.  [✨ 자동 생성] 보라색 배너 → 모달
         · skip_holidays / mark_leaves / overwrite / clear_first 옵션
         · [🔍 미리보기] → INSERT N건 / UPDATE M건 확인
         · [✨ 적용]
Step 5.  좌측 워커 list → 부담 막대 + 평균선 + 편차 색상 점검
         · ⚠ 과로 / ⬇ 부족 워커 발견 → 작성 모드에서 일자 토글로 조정
Step 6.  KPI 5번째 타일 [균형도] — 양호 / 주의 / 위험
         · 위험이면 분석 드로어 [📊 분석] 열어 인당 비교
Step 7.  [📋 매트릭스] 모드 → [👀 빈자리 강조] 토글
         · 빈 셀 빨간 점선 → 우클릭 quick action 또는 클릭 picker 로 채움
Step 8.  [🔄 시프트 교체] 토글 → 두 셀 클릭으로 swap
Step 9.  [📅 날짜별] 모드로 일자별 인원 점검
Step 10. 헤더 [⋯ 더보기] → [✓ 공지됨으로 변경]
Step 11. [⚡ 배포] → 모달 → 채널 선택 (manual/link/잔디/이메일) → 발송 기록
```

### 1-C. 결근 / 시프트 교체 즉석 처리

```
A. 직원이 "이 날 못 나옴" 사전 신청 (PR-2Y):
   매니저 [⋯] 버튼에 빨간 배지 N건
   → 클릭 → 요청 list (Phase 추후) 또는 매트릭스에서 해당 셀 직접 처리

B. 갑작스런 결근:
   매트릭스 모드에서 해당 셀 우클릭 → [💤 휴무] 또는 [🌅 오전반차]
   → 빠른 대체: [🔄 시프트 교체] 토글 → 다른 워커와 swap

C. 본인 연차 등록:
   설정 → 📋 직원 휴가 탭 → [+ 연차 등록]
   → 워커/기간/종류/반차 → 잔여 부족 시 경고
   → 자동 생성 다시 돌리면 해당 일자 자동 휴무 적용
```

## 페르소나 2 — 직원

### 2-A. 영구 토큰 링크 받음 (입사 후 1회)

```
Step 1. 매니저가 RideEmployees → 본인 카드 → [+ 영구 링크 발급]
Step 2. URL 잔디/카톡으로 1회 받음 → 휴대폰 북마크
Step 3. /CallScheduler/e/<token> 비로그인 진입
        · 본인 시간표만 (다른 직원 일정은 같은 날 동료만 노출)
        · published 스케줄만 (초안은 안 보임)
```

### 2-B. 매월 스케줄 확인

```
Step 1. 북마크 클릭 → 마이페이지
Step 2. "🆕 새 공지" 펄스 배지 (7일 이내 공지) — 매니저가 공지하면 자동
Step 3. KPI 4타일 — 근무일수 / 총시간 / 야간 / 반차·F
Step 4. 캘린더 → 본인 근무일만 컬러 강조
        · 캘린더 셀 클릭 → 같은 날 동료 모달 (본인은 진한 테두리 + 👤)
Step 5. 상세 시간표 list — 날짜·요일·시간·구분
Step 6. [📥 캘린더 다운로드] → iCal(.ics) 파일
        · 휴대폰 캘린더 앱에 import → 본인 일정 자동 동기화
```

### 2-C. 시프트 교체 요청

```
Step 1. 마이페이지 시간표 list → 해당 날짜 우측 [🙋 교체 요청]
Step 2. 모달 → 사유 입력 → [🙋 매니저에게 신청]
Step 3. cs_swap_requests INSERT (status='pending')
Step 4. 매니저 [⋯] 빨간 배지에 카운트 N건 표시
Step 5. 매니저 처리:
        · approved → 매트릭스에서 swap 또는 결근 처리
        · rejected → 직원에게 알림 (Phase 추후)
        · canceled → 직원이 취소 가능 (Phase 추후)
```
