## 3. 매트릭스 셀 디자인

### 크기
- 높이: **32px** (이전 24px → 확대)
- 너비: 셀 td 최소 **56px** (이전 44px)
- 셀 안 폰트: **12px** (이전 11px)

### 워커 색상
- 14색 토큰 (TONE_BG/BORDER/TEXT)
- 매트릭스 셀: tone 진한 보더 + 옅은 배경
- 워커별 시각 식별 강화

### 상태 표시 우선순위
1. 🔒 manual_lock (좌상단 작은 아이콘)
2. 🛌 회피 (셀 회색 + 회피일 행에서 별도 시각화)
3. ⏳ 회피 대기
4. 휴 / 오반 / 오후반 / F 등 special_code

### Phase D 색상 layer (NEW)
- **희망 요일 매치**: 옅은 녹색 보더 또는 배경 hint (rgba(34,197,94,0.08))
- **비선호 요일 매치**: 옅은 빨강 보더 또는 배경 hint (rgba(239,68,68,0.08))
- **외부 cycle 근무 phase**: 회색 줄 패턴 (이미 워커별 행에 표시 — 셀에는 옅은 회색 hint)

## 4. 페이지 레이아웃 통일

### 매트릭스 페이지 (`/CallScheduler/[id]`)
- 헤더: 배지 (status / source) + 날짜 + 액션
- KPI strip (5 카드)
- 매트릭스 (시프트 × 일자) + 외부 cycle 행 + 회피일 행
- 사이드 드로어: 분석 / 배포 / 직원 요청

### 설정 페이지 (`/CallScheduler/settings`)
- 6 탭 균등 폭
- 탭 안 콘텐츠: max-width 1400px
- 폼: 좌·우 2 컬럼 (1fr 1fr) — 모바일 시 1 컬럼

### 모달
- 좌상단 헤더 + 우상단 닫기 (×)
- 본문: scrollable (max-height 85vh)
- 하단: 취소 / 액션 버튼 (BTN.md 이상)
