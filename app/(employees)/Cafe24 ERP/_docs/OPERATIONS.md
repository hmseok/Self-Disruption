## 6. 기술 운영 사실 (확정)

```
✅ DB 종류: MariaDB 10.1.13 (Distrib 10.0.21)
✅ 차밍 charset: utf8 (utf8mb3 가능성 — 백업 헤더 SET NAMES utf8)
✅ TIME_ZONE: '+00:00' (UTC) — 한국 시간 변환 필요 시 +9시간
✅ 자격증명: .env.local 의 CAFE24_DB_* (5개 키)
✅ 호스트: PHP 측 localhost (서버 내부) / FMI 측 외부 IP 필요
✅ 포트: ❓ 3306 가정 (확인 필요)
✅ DB 종류: 단일 DB (멀티 DB 패턴 없음)
✅ 외부 API 통합: 알리고 SMS / 카카오 / 다음 / 잔디 / 주소
✅ 동시 사용: PB 데스크톱 + PHP 웹 + (이번 추가) FMI ERP
```
