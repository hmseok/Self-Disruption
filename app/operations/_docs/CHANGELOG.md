## 2026-07-05 PR-AUTO-LINK — 확실한 입금 자동 연결 승격 (사용자 명시 「맞는 건 자동연결」)
- 매처 자동 적용 축 2개 추가: ① 입금자명에 **차량번호 전체**(예: 69서0895) + 그 차량 대차건 유일 → HIGH 자동 ② 입금자명 == **고객명** 정규화 일치(한글 2~5자) + 대차건 유일 → MEDIUM 자동.
- 애매한 건(다수 후보·뒤4자리만·보험사 mismatch)만 LOW → 청구 탭 검수 패널에 남음.
