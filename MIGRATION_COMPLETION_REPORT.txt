AUTHENTICATION IMPLEMENTATION
================================================================================

Helper Function (added to all 9 files):
────────────────────────────────────────
async function getAuthHeader(): Promise<Record<string, string>> {
  const { data: { session } } = await supabase.auth.getSession()
  const token = session?.access_token
  return token ? { Authorization: `Bearer ${token}` } : {}
}

Usage Pattern in fetch() calls:
──────────────────────────────
const res = await fetch('/api/endpoint', {
  method: 'POST|PATCH|DELETE',
  headers: { 
    'Content-Type': 'application/json', 
    ...(await getAuthHeader()) 
  },
  body: JSON.stringify(data)
})

API Route Authentication (in all new routes):
──────────────────────────────────────────────
1. Extract Bearer token from Authorization header
2. Decode JWT to extract user ID
3. Query profiles table to get company_id and role
4. Filter all queries by company_id
5. Return 401 for unauthenticated requests

ERROR HANDLING PATTERNS
================================================================================

Frontend (fetch calls):
───────────────────────
const res = await fetch('/api/endpoint', { ... })
const json = await res.json()
if (!res.ok) throw new Error(json.error)

Backend (API routes):
─────────────────────
try {
  // validation
  // query execution
  return NextResponse.json({ data: result, error: null })
} catch (e: any) {
  return NextResponse.json({ error: e.message }, { status: 500 })
}
